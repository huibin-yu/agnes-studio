# AI 生成效果测试提示词

生成时间：2026-06-20 12:41:22 CST

用途：用于在 Agnes Studio 前端页面测试图片生成和视频生成效果。每条提示词都配了推荐参数，便于对比模型在主体一致性、细节、构图、光影、运动稳定性上的表现。

## 使用方式

1. 启动前端：`http://127.0.0.1:3000`
2. 图片测试进入：`/generate/image`
3. 视频测试进入：`/generate/video`
4. 复制下方提示词到输入框。
5. 按推荐参数选择尺寸、风格、帧数和 FPS。
